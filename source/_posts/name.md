常用的项目目录文件命名
<!--more-->

# config
> 配置文件

# static
> 静态资源

# src,source
> 源代码

# examples，demo
> 示例

# docs
> 文档

# plugins
> 插件

# lib
> library,库文件

# component
> 组件

# dist
> distribution,打包编译后的文件
>> less,sass,typescript,es6

# test,tests,__test__
> 测试文件

## benchmarks
> 测试版本的性能变化

## units,spec
> 单元测试

## e2e
> 端对端测试

# build,script
> 构建脚本

# utils,tools,helpers
> 工具代码

# models,views,controllers,middlewares
> MVC，中间件

# router
> 路由
 
# server
> 服务端代码

# adapters
> 适配器模式

# legacy
> 兼容历史版本或者浏览器的代码

# bin
> 命令脚本，命令行工具

# common
> 公用文件

# packages
> 打包多个npm包,减小体积

# core
> 核心文件

# misc
> 杂项
